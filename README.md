# Clearra Privacy Policy

Static privacy policy page for Clearra by Self Meta Labs.

Target GitHub Pages URL:

```text
https://selfmetalabs.github.io/clearra-privacy-policy/
```

## Publish With GitHub Pages

1. Create a public GitHub repository named `clearra-privacy-policy`.
2. Upload `index.html` to the repository root.
3. Go to repository `Settings > Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save and wait for GitHub Pages to publish.
7. Use the published URL in Google Play Console's `Privacy Policy URL` field.
